go-libre-storage
=========

Cloud storage and file share platform writen in [Go](https://golang.org) (Golang).

## Build

Windows
```
> mingw32-make && mingw32-make build
```

Unix
```
> make && make build
```

## Run development instance

```
> HOST="localhost" ./bin/go-libre-storage
```
